# Facebook Login UI Clone
This project is a simple UI design of the Facebook login page recreated using Figma and basic HTML & CSS.
## Purpose
- Practice UI design skills
- Improve layout and alignment understanding
- Learn frontend structure basics
## Tech Used
- HTML
- CSS
- Figma (for design reference)
## Note
This is only a UI design project, not a functional login system.
## Status
Completed as a beginner practice project.
